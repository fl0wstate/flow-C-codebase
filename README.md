## Getting Started

### Prerequisites

- A C compiler (e.g., gcc, clang, MSVC)
- `make` (for Unix-like systems)
- Git (optional, for version control)

### Building the Project

Compile using `make`:
```sh
make
```

### Running

After building, run the executable:
```sh
./binary
```
